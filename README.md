# e-Plant Shopping Application

This is a React and Redux Toolkit based e-commerce application for purchasing houseplants.

## Features
- Landing page with company information
- Product listing page with categorized plants
- Add to Cart functionality using Redux
- Shopping cart with increment, decrement, and delete options
- Displays total items and total cost dynamically

## Technologies Used
- React
- Redux Toolkit
- React Router DOM
- CSS

## How to Run the Project
1. Clone the repository
2. Run the following commands:

npm install  
npm start
